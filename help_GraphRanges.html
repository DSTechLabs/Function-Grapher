Graph Ranges
X/Y-Axis Digital Inputs: to
X and Y axis ranges can be defined manually by entering values in the input fields.
For each axis, the first field is the "minimum" value and the second field is the "maximum" value.

All functions are replotted when focus leaves a field or the [Enter] key is pressed.
You may use the [Tab] key to move the focus through the fields.

The initial default axis ranges are from -10.5 to 10.5.
This button will reset the ranges back to their default values.

You may set new default values by pressing the button.

This button will set the current axis ranges as the new default set.
Then pressing the button will reset the ranges back to these values.
